## 中国茶的悠久历史

茶的起源可以追溯到数千年前的古代中国。中国人最先发现了茶叶的奇妙之处，这一发现从此成为中华文化不可分割的一部分。

### 历史年表

| 时代 | 时间 | 里程碑 |
|------|------|--------|
| 上古传说 | 约公元前2737年 | 神农尝百草，茶叶偶然落入沸水中被发现 |
| 汉代 | 前206–公元220年 | 茶作为药用和贡品，出现最早的文字记载 |
| 唐代 | 618–907年 | 陆羽著《茶经》，世界第一部茶学专著 |
| 宋代 | 960–1279年 | 点茶法盛行，斗茶之风兴起 |
| 明代 | 1368–1644年 | 散茶冲泡取代末茶，六大茶类雏形初现 |
| 清代 | 1644–1912年 | 丝绸之路与海上茶叶贸易，普洱、乌龙茶声名鹊起 |
| 近现代 | 1900年至今 | 中国茶文化复兴，功夫泡法广泛流行 |

## 中国六大茶类

中国拥有种类繁多的茶叶，每种茶都有其独特的品性和风味。六大茶类按照氧化程度和加工工艺分类：

### 氧化程度光谱

```
  不发酵                                    全发酵          后发酵
  ◄─────────────────────────────────────────────────────►   ◄──────────►

  ██░░░░░░░░  绿茶  Green Tea    （杀青，不氧化）
  ██░░░░░░░░  白茶  White Tea    （自然萎凋，微氧化）
  ███░░░░░░░  黄茶  Yellow Tea   （杀青 + 闷黄）
  ████████░░  乌龙  Oolong Tea   （部分氧化，10–70%）
  ██████████  红茶  Black Tea    （全氧化）
                                                            ██████████
                    黑茶  Dark Tea    （微生物后发酵）
```

| 茶类 | 英文名 | 氧化程度 | 核心特点 |
|------|--------|---------|---------|
| [绿茶](ChineseTea/Process/GreenTea/README_cn.md) | Green Tea | 不发酵 | 清新、鲜爽 |
| [白茶](ChineseTea/Process/WhiteTea/README_cn.md) | White Tea | 微发酵 | 清淡、甘甜 |
| [黄茶](ChineseTea/Process/YellowTea/README_cn.md) | Yellow Tea | 轻发酵 | 醇和、柔滑 |
| [乌龙茶](ChineseTea/Process/OolongTea/README_cn.md) | Oolong Tea | 半发酵（10–70%） | 层次丰富、香气复杂 |
| [红茶](ChineseTea/Process/BlackTea/README_cn.md) | Black Tea | 全发酵 | 浓郁、醇厚 |
| [黑茶](ChineseTea/Process/DarkTea/README_cn.md) | Dark Tea | 后发酵 | 陈韵、厚重 |

> **注：** 普洱茶是云南生普洱（工艺接近绿茶，仍有争议）和熟普洱（属于黑茶）的统称。

### 六大茶类工艺一览

| 茶类 | 主要工序 | 独特工艺 |
|------|---------|---------|
| 绿茶 | 采摘 → **杀青** → 揉捻 → 干燥 | 杀青阻止所有氧化 |
| 白茶 | 采摘 → **萎凋** → 干燥 | 最简工艺，不经杀青 |
| 黄茶 | 采摘 → 杀青 → 揉捻 → **闷黄** → 干燥 | 闷黄增添醇和感 |
| 乌龙茶 | 采摘 → 萎凋 → **摇青** → 杀青 → 揉捻 → 干燥 | 摇青控制部分氧化程度 |
| 红茶 | 采摘 → 萎凋 → 揉捻 → **全发酵** → 干燥 | 完全酶促氧化 |
| 黑茶 | 采摘 → 杀青 → 揉捻 → **渥堆** → 干燥 | 微生物参与发酵，持续数周至数月 |

## 中国茶的冲泡艺术

中国传统茶道是一门真正的艺术，是水、叶与时间的精妙配合，体现了敬意、专注与欣赏。正确冲泡和品饮中国茶，需要注意：

- **水质** — 以山泉水或过滤水为佳
- **茶具** — 根据茶类选用盖碗、紫砂壶或玻璃杯
- **水温** — 从70°C（绿茶）到100°C（黑茶）不等
- **浸泡时间** — 因茶类和个人口味而异

### 冲泡速查表

| 茶类 | 水温 | 茶水比 | 浸泡时间 | 推荐茶具 |
|------|------|--------|---------|---------|
| 绿茶 | 75–85°C | 3克/150ml | 1–3分钟 | 玻璃杯 |
| 白茶 | 85–95°C | 3–5克/150ml | 1–2分钟 | 盖碗 |
| 黄茶 | 80–85°C | 3克/150ml | 1–2分钟 | 玻璃杯 |
| 乌龙茶 | 90–100°C | 5–8克/100ml | 10–15秒 | 盖碗/紫砂壶 |
| 红茶 | 85–95°C | 3–5克/150ml | 2–3分钟 | 盖碗/茶壶 |
| 黑茶 | 95–100°C | 5–8克/100ml | 10–15秒 | 紫砂壶 |

## 中国茶的健康益处

除了令人愉悦的口感，中国茶还具有众多潜在的健康益处。研究表明，茶中的抗氧化物质有助于改善身体健康。茶也是放松身心的良伴，是忙碌生活中片刻的宁静。它不仅能愉悦味蕾，更能滋养身心。
