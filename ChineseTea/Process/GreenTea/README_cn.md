# 绿茶 (Green Tea)

[English Version](README.md) | [← 返回首页](../../../README_cn.md)

## 概述

绿茶是中国产量最大、饮用最广的茶类。绿茶属于**不发酵茶**，其关键工序是"杀青"——通过高温阻止茶叶氧化，保留鲜叶的清新风味。

## 制作工艺

绿茶最重要的工序是**杀青**，通过加热使酶失活，阻止氧化。根据杀青方式的不同，绿茶分为四大类：

| 类型 | 英文 | 工艺特点 | 代表茶品 |
|------|------|---------|---------|
| 炒青 | Pan-fired | 在热锅中翻炒杀青 | 龙井、碧螺春 |
| 烘青 | Baked | 烘箱干燥 | 黄山毛峰、太平猴魁 |
| 蒸青 | Steamed | 蒸汽杀青，保持翠绿 | 恩施玉露 |
| 晒青 | Sun-dried | 日光晒干 | 云南晒青毛茶（生普原料） |

**完整工序：** 采摘 → 杀青 → 揉捻 → 干燥

## 名优绿茶

- **龙井** — 产自浙江杭州，扁平光滑，板栗香气。
- **碧螺春** — 产自江苏洞庭，条索紧卷，花果香浓。
- **黄山毛峰** — 产自安徽黄山，细嫩微甜，兰花香气。
- **太平猴魁** — 产自安徽，叶片扁平硕大，独特兰香。
- **信阳毛尖** — 产自河南，细紧圆直，清爽甘醇。

## 冲泡指南

- **水温：** 75–85°C（切勿使用沸水）
- **茶水比：** 3克/150毫升
- **浸泡时间：** 1–3分钟
- **茶具：** 玻璃杯或盖碗（便于观赏叶片舒展）

## 健康益处

绿茶富含儿茶素和茶多酚，具有抗氧化、促进代谢的功效。适量的咖啡因和茶氨酸有助于提神醒脑。
