# 黄茶 (Yellow Tea)

[English Version](README.md) | [← 返回首页](../../../README_cn.md)

## 概述

黄茶是六大茶类中最为稀有和小众的品类。它的制作工艺与绿茶非常相似，但多了一道关键步骤——**闷黄**。这一步骤使茶汤更加醇和柔滑，减少了绿茶的苦涩感。

## 制作工艺

**完整工序：** 采摘 → 杀青 → 揉捻 → **闷黄** → 干燥

闷黄是黄茶独有的工序，将杀青后尚带温热和水分的茶叶用布或纸包裹，进行缓慢而轻微的氧化。这一过程：
- 减少绿茶的青草气
- 形成更加醇和、甘甜的口感
- 使茶叶和汤色呈现独特的黄色

## 名优黄茶

- **君山银针** — 产自湖南岳阳，全由芽头制成，中国十大名茶之一。
- **蒙顶黄芽** — 产自四川蒙顶山，芽叶细嫩，甘甜带板栗香。
- **霍山黄芽** — 产自安徽霍山，清鲜醇和，回味清爽。

## 市场现状

与其他茶类相比，黄茶的市场规模较小。闷黄工序对技术要求高、耗时费力，使得黄茶产量有限、价格较高。然而，正因其稀有，黄茶深受茶叶爱好者的珍视。

## 冲泡指南

- **水温：** 80–85°C
- **茶水比：** 3克/150毫升
- **浸泡时间：** 1–2分钟
- **茶具：** 玻璃杯或盖碗
