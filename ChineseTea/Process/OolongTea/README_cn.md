# 乌龙茶 (Oolong Tea)

[English Version](README.md) | [← 返回首页](../../../README_cn.md)

## 概述

乌龙茶是工艺上最为复杂的一类茶，属于**半发酵茶**，发酵程度从10%到70%不等。这一宽泛的范围赋予了乌龙茶极其丰富的风味多样性——从清香花韵到浓醇焙火，变化万千。

## 制作工艺

**完整工序：** 采摘 → 萎凋 → 摇青/做青 → 杀青 → 揉捻 → 干燥/焙火

乌龙茶独有的关键工序是**摇青（做青）**，通过轻轻摇动和碰撞茶叶边缘，促进部分氧化。制茶师通过控制摇青的次数、力度和时间来把握发酵程度，这一过程高度依赖经验和技艺。

## 主要产区与品种

### 闽北（福建北部）
- **大红袍** — 武夷岩茶之王，重焙火，矿物感与花香交织。
- **肉桂** — 武夷岩茶名品，以桂皮般的辛香著称。
- **水仙** — 滋味醇厚顺滑，兰花香气。

### 闽南（福建南部）
- **铁观音** — 中国最著名的茶之一。清香型花韵悠长，浓香型醇厚回甘。

### 广东
- **凤凰单丛** — 以模拟天然花果香著称：蜜兰香、杏仁香、栀子香等。

### 台湾
- **冻顶乌龙** — 中度焙火，滋味甘醇顺滑。
- **东方美人** — 重度发酵，蜜甜带果香。
- **阿里山乌龙** — 高山茶，清香淡雅。

## 冲泡指南

- **水温：** 90–100°C（焙火茶用更高水温）
- **茶水比：** 5–8克/100毫升（功夫泡法）
- **浸泡时间：** 首泡10–15秒，逐泡递增
- **茶具：** 紫砂壶或盖碗
- **泡法：** 推荐功夫泡法，多次冲泡感受风味的层次变化。
