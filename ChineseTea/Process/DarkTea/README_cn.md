# 黑茶 (Dark Tea)

[English Version](README.md) | [← 返回首页](../../../README_cn.md)

## 概述

黑茶属于**后发酵茶**，会有微生物参与发酵，使茶叶本身发生较大的化学变化，形成独特的陈韵和厚重口感。好的黑茶，茶汤应当透亮清澈——汤色的通透是品质的标志。

## 制作工艺

**完整工序：** 采摘 → 杀青 → 揉捻 → **渥堆发酵** → 干燥 → （可选：压制成型）

核心工序是**渥堆**，将茶叶堆放在温暖潮湿的环境中，促进微生物发酵。这一过程可能持续数周到数月，是黑茶区别于其他茶类的关键。

## 名优黑茶

| 茶品 | 产区 | 特点 |
|------|------|------|
| 熟普洱 | 云南 | 醇滑厚重，越陈越香 |
| 安化黑茶 | 湖南 | 茯砖茶有独特的"金花"（有益菌群） |
| 老青砖 | 湖北 | 传统砖茶，厚实醇和 |
| 六堡茶 | 广西 | 槟榔香，滑爽甘甜 |
| 四川边茶 | 四川 | 历史上销往西藏，浓厚耐泡 |

## 陈化与存储

黑茶是少数**越陈越好**的茶类。存放得当的黑茶可以保存数十年，滋味会随时间变得更加醇厚、深沉、复杂。存储要求：
- 干燥、通风良好的环境
- 远离异味
- 稳定的室温

## 冲泡指南

- **水温：** 95–100°C（沸水）
- **茶水比：** 5–8克/100毫升
- **浸泡时间：** 首泡10–15秒，逐泡递增
- **茶具：** 紫砂壶或盖碗
- **提示：** 第一泡先快速洗茶（润茶），唤醒茶叶。
