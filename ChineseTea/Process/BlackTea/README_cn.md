# 红茶 (Black Tea)

[English Version](README.md) | [← 返回首页](../../../README_cn.md)

## 概述

红茶是世界上最受欢迎、饮用最广的茶类之一，属于**全发酵茶**，以浓郁醇厚的口感和红亮的汤色著称。英文中称为"Black Tea"（黑茶），而中文根据其汤色称之为"红茶"。

## 制作工艺

**完整工序：** 采摘 → 萎凋 → 揉捻 → **全发酵** → 干燥

红茶与其他茶类的核心区别在于**全发酵**工序，揉捻后的茶叶在温暖潮湿的环境中充分氧化，直至叶色转为深铜红色。

## 按工艺分类

红茶按照加工方式分为三大类：

| 类型 | 英文 | 特点 | 代表茶品 |
|------|------|------|---------|
| 小种红茶 | Small-leaf | 起源于武夷山，传统松烟熏干 | 正山小种 |
| 工夫红茶 | Gongfu Black Tea | 制作精细考究 | 祁门红茶、滇红 |
| 红碎茶 | CTC / Broken | 机器切碎，浸出快 | 用于茶包和拼配 |

## 名优红茶

- **正山小种** — 产自福建武夷山，世界上最早的红茶。传统工艺有独特的松烟香。
- **祁门红茶** — 产自安徽祁门，世界三大高香红茶之一，似果似蜜的"祁门香"。
- **滇红** — 产自云南，以大叶种制成，茶汤金黄，蜜甜醇厚。
- **金骏眉** — 福建顶级红茶，全芽头制作，甘甜蜜香。

## 冲泡指南

- **水温：** 85–95°C
- **茶水比：** 3–5克/150毫升
- **浸泡时间：** 2–3分钟（大杯泡法）或5–10秒（功夫泡法）
- **茶具：** 盖碗、玻璃杯或茶壶
