# 花茶 — 再加工茶 (Flower-Scented Tea)

[English Version](README.md) | [← 返回首页](../../README_cn.md)

## 概述

茉莉花茶等花茶是中国非常著名的茶类，但它**不属于六大茶类**，而属于**再加工茶**，因为在制作过程中加入了茶叶以外的原料——茉莉花等花朵。

## 什么是"再加工"？

花茶以茶坯（通常为绿茶，也有白茶或乌龙茶）为基底，通过与鲜花反复**窨制**（将茶叶与花朵层层交替堆放），使茶叶充分吸收花香。窨制次数越多，花香越浓郁。

## 常见花茶

- **茉莉花茶** — 最受欢迎的花茶，以绿茶为茶坯，窨入茉莉花香。福建、广西为主要产区。
- **玫瑰花茶** — 以玫瑰花瓣窨制或拼配，花香甜美。
- **桂花茶** — 以桂花窨制，香气清雅甜润。

## 冲泡指南

- **水温：** 80–90°C
- **茶水比：** 3克/150毫升
- **浸泡时间：** 2–3分钟
- **茶具：** 玻璃杯或瓷杯（便于品赏香气）

当然，花茶也非常好喝！
