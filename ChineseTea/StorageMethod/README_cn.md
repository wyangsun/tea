# 茶叶存储方式 (Tea Storage Methods)

[English Version](README.md) | [← 返回首页](../../README_cn.md)

## 按存储形态分类

茶叶按物理形态可分为**紧压茶**和**散茶**两大类。

## 紧压茶

紧压茶是将茶叶压制成各种形状，便于储存、运输和陈化。常见形态包括：

| 形态 | 说明 | 常见茶品 |
|------|------|---------|
| 饼茶 | 压成扁平圆饼状 | 普洱茶、福建白茶 |
| 砖茶 | 压成长方形砖块 | 安化黑茶、老青砖 |
| 沱茶 | 压成碗状或窝状 | 云南普洱、下关沱茶 |
| 其他形状 | 大小各异，便于携带分装 | 小饼、龙珠等 |

压制的主要目的：
- **便于长期陈化** — 普洱和白茶常压成饼状，利于缓慢转化
- **便于储存运输** — 紧凑形态节省空间
- **方便分装冲泡** — 小型紧压茶易于掰取

## 散茶

散茶以自然松散的形态存储。大多数绿茶、黄茶和乌龙茶以散茶形式销售。散茶存储要点：

- **密封容器** — 隔绝空气，防止氧化
- **阴凉避光** — 远离高温和阳光直射
- **远离异味** — 茶叶极易吸附周围气味
- **低湿度** — 潮湿会导致发霉变质
- **冷藏保存** — 细嫩绿茶建议密封后冷藏（需防潮）

## 各茶类存储建议

| 茶类 | 最佳存储方式 | 保质期 |
|------|------------|--------|
| 绿茶 | 密封冷藏 | 6–18个月 |
| 白茶 | 干燥通风，可陈化 | 数年至数十年 |
| 黄茶 | 密封阴凉 | 6–12个月 |
| 乌龙茶 | 密封；焙火型更稳定 | 1–3年（清香）；多年（焙火） |
| 红茶 | 密封常温 | 1–3年 |
| 黑茶 | 干燥通风，越陈越好 | 数十年 |
