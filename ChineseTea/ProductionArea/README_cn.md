# 中国茶叶产区 (Tea Production Regions)

[English Version](README.md) | [← 返回首页](../../README_cn.md)

## 概述

中国有众多茶叶产区，南方各省基本都有茶叶种植。目前比较著名的产区包括**福建、云南、四川、浙江、安徽**等。

## 主要茶区

| 省份 | 名优茶品 | 涉及茶类 |
|------|---------|---------|
| 福建 | 铁观音、大红袍、正山小种、福鼎白茶、茉莉花茶 | 乌龙、红、白、花 |
| 云南 | 普洱茶（生普/熟普）、滇红 | 黑、红、绿 |
| 浙江 | 龙井、安吉白茶 | 绿 |
| 安徽 | 黄山毛峰、祁门红茶、太平猴魁、六安瓜片 | 绿、红 |
| 四川 | 蒙顶甘露、蒙顶黄芽、四川边茶 | 绿、黄、黑 |
| 湖南 | 君山银针、安化黑茶 | 黄、黑 |
| 江苏 | 碧螺春 | 绿 |
| 广东 | 凤凰单丛 | 乌龙 |
| 广西 | 六堡茶 | 黑 |
| 河南 | 信阳毛尖 | 绿 |
| 湖北 | 恩施玉露、老青砖 | 绿、黑 |

## 以地区命名的茶叶

许多中国茶以产地命名：

- **滇红** — "滇"是云南的古称，即云南红茶。
- **闽红** — "闽"是福建的古称，即福建红茶。
- **普洱茶** — 以云南普洱市命名。
- **岩茶** — 产自福建武夷山岩石地貌的乌龙茶。
- **六堡茶** — 以广西六堡镇命名。
